## 📌 Project Overview

This project demonstrates an end-to-end **Big Data Analytics pipeline** using **PySpark** and **Power BI** to analyze an e-commerce dataset and generate business insights.

The objective was to process large-scale transactional data, perform data cleaning and transformation using PySpark, calculate business KPIs, and build interactive dashboards that support data-driven decision-making.

Unlike traditional analytics projects that rely only on Pandas, this project uses **PySpark**, making it scalable for large datasets and introducing concepts used in modern data engineering and analytics.

---

# 🎯 Business Problem

E-commerce organizations generate large volumes of transactional data every day. Traditional data processing tools become inefficient when data size increases, making it difficult to identify sales trends, profitable products, customer behavior, and regional performance.

This project addresses these challenges by building a scalable analytics workflow capable of processing large datasets and delivering interactive business insights.

# 📌 Key Business Findings

* Technology generated the highest sales and profit.
* Consumer customers contributed the highest revenue.
* West region achieved the highest profitability.
* Central region generated relatively strong sales but the lowest profit due to higher discounts.
* Phones generated the highest sales.
* Copiers generated the highest profit.
* Average Discount: **15.63%**
* Profit Margin: **12.47%**

---

# 💡 Business Recommendations

* Review discount strategy in the Central region.
* Continue investment in Technology products.
* Promote high-margin products such as Copiers.
* Optimize pricing for low-margin products.
* Focus marketing efforts on the Consumer segment.
